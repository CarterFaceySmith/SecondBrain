# Arrays

A container which holds a fixed (normally) number of items of the same type.

Each element in the array has a position index.

#### Basic Operations
- Traversal
- Insertion
	- insertEnd (O(1))
	- insertMid (O(n))
- Deletion
	- removeEnd (O(1))
	- removeMid (O(n))
- Search
- Update
